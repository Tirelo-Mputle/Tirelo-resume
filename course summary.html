Html tags

Html tags and their uses

All of these tags have closing and opening tags.

  • html: encloses the whole html page and tell the browser that the content it is reading is html
  • head: is at the top of the page. It contains details about the page that are passed on to the browser.
  • meta: contains other more specific data that the browser needs. It tells the browser the description of the page, the type of encoding ...
  • title: contains the title of the page that will appear in the browser tab of that page.
  • body: the main content of the page goes in here. Everything that appears on the screen, that is html is in here.

  • h1-h6: heading sizes
  • p: paragraphs
  • em and i: em emphasis text and is preffered over i which writes it in italics
  • strong and bold: strong emphasis the text while bold only boldens it with no context.
  • ul: unordered list
  • ol: ordered list
  • li: contains the actual list item

These html tags don't have closing tags.

  • hr: creates a horizontal line.
  • br: creates a space.
  • img: contains images
  • a: anchor tag. Allow text or image to become clicable and link to something.
